# DermaAI - Skin Condition Detection

AI-powered skin condition detection platform built using Python, Flask, and TensorFlow.

### Dataset
The dataset used for training this model is available on Kaggle:
[Click here for Kaggle Dataset](https://www.kaggle.com/datasets/pacificrm/skindiseasedataset)

### Project Overview
This platform detects skin conditions using an AI model. It allows users to upload images and get predictions based on the trained model logic.
